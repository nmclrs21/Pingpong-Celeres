Title: Ping Pong Bounce Mania
Developer: Naomi Celeres
Tool: Python (Pygame)

Description:
This is a pong-style game where the player competes against AI or another player.
The goal is to reach 3 points first.

Features:
- AI mode and PvP mode
- Paddle movement using keyboard
- Ball collision detection
- Scoring system
- Win condition at 3 points# Pingpong-Celeres
